### Props

| Property     | Attribute     | Description | Type                                                                                | Default     |
| ------------ | ------------- | ----------- | ----------------------------------------------------------------------------------- | ----------- |
| `color`      | `color`       |             | `"danger" \| "default" \| "ghost" \| "info" \| "primary" \| "success" \| "warning"` | `"default"` |
| `disabled`   | `disabled`    |             | `boolean`                                                                           | `false`     |
| `icon`       | `icon`        |             | `string`                                                                            | `""`        |
| `loading`    | `loading`     |             | `boolean`                                                                           | `false`     |
| `selected`   | `selected`    |             | `boolean`                                                                           | `false`     |
| `size`       | `size`        |             | `"large" \| "medium" \| "small"`                                                    | `"medium"`  |
| `suffixIcon` | `suffix-icon` |             | `string`                                                                            | `""`        |
| `variant`    | `variant`     |             | `"default" \| "neo" \| "outline" \| "solid" \| "text"`                              | `"default"` |


### Events

| Event         | Description | Type                                                     |
| ------------- | ----------- | -------------------------------------------------------- |
| `buttonClick` |             | `CustomEvent<{ event: MouseEvent; selected: boolean; }>` |

### 1. Basic buttons (default state)

Shows a minimalist button with an appearance that can be adjusted based on `size`, `variant` and `color`.

<air-previewer size="large">
<air-button size="medium" variant="default" color="default">
  Primary Button
</air-button>
</air-previewer>

### 2. Buttons of different colours

Displays buttons in different colours, covering `primary`, `success`, `info`, `warning`, `danger`, and `ghost`.

<air-previewer size="large">
<>
<air-button color="primary">primary</air-button>
<air-button color="success">success</air-button>
<air-button color="info">info</air-button>
<air-button color="warning">warning</air-button>
<air-button color="danger">danger</air-button>
<air-button color="ghost">ghost</air-button>
</>
</air-previewer>

### 3. Buttons of different sizes demonstrate the different effects of the 
`size` attribute. You can use `small`, `medium`, and `large` to demonstrate different button sizes.

<air-previewer size="large">
<>
<air-button size="small">small Button</air-button>
<air-button size="medium">medium Button</air-button>
<air-button size="large">Large Button</air-button>
</>
</air-previewer>

### 4. Buttons with icons
Displays buttons with icons. Use the `icon`,`suffix-icon` attributes to add prefix and suffix icons to buttons.

<air-previewer size="large">
<>
<air-button icon="⭐">图标按钮</air-button>
<air-button suffix-icon="✓">后缀图标按钮</air-button>
<air-button icon="🔄" suffix-icon="✓">双图标按钮</air-button>
</>
</air-previewer>

### 5. Different styles of buttons
show how buttons look when they are in button style mode, using the `solid`, `outline`, `text`, `neo`, and `default` attributes.

<air-previewer size="large">
<>
<air-button variant="solid">solid</air-button>
<air-button variant="outline">outline</air-button>
<air-button variant="text">text</air-button>
<air-button variant="neo">neo</air-button>
<air-button variant="default">default</air-button>
</>
</air-previewer>

### 6. Loading, disabled, selected state display
Display buttons in the loading, disabled, or selected state using the `loading`, `disabled`, `icon`, `selected`, and other attributes.

<air-previewer size="large">
<>
<air-button loading>loading</air-button>
<air-button disabled>disabled</air-button>
<air-button disabled icon="❌">icon</air-button>

</>
</air-previewer>
